##  About the Project

This is a web-based Online EAMCET Examination Portal developed using PHP and MySQL.

The system was originally designed and implemented for use in my college to conduct entrance-style mock examinations in a structured and automated manner.

It solves real-world problems such as:

- Manual evaluation delays
- Paper-based exam management difficulties
- Lack of centralized question control
- No automated timer enforcement
- Manual result calculation errors

##  Key Features

-  30 Questions (10 per subject in demo)
-  45-Minute Overall Timer
-  Question Navigation & Review System
-  Automatic Score Calculation
-  Subject-wise Question Segmentation
-  Image-Based Questions Support
-  Secure Admin Panel for Question Management

##  Admin Dashboard

The system includes an Admin Panel that allows:

- Adding new questions
- Uploading question & option images
- Managing subjects
- Controlling exam structure

(Admin credentials are not publicly shared.)

##  Real-World Implementation

This project was developed to simulate a real competitive examination environment similar to state-level entrance exams.

It demonstrates:

- Backend session management
- Database-driven question rendering
- Timer control logic
- Secure result computation
- Role-based access (Admin & Student)
